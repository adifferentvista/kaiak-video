## Prompt Examples

### 1. The Kinetic Marketing
**Link:** https://www.remotion.dev/prompts/the-kinetic-marketing
**Tool:** Gemini | **Model:** k2.5

A high-velocity kinetic typography promo for a modern developer tool.

**Prompt:** Visual Style: Premium "Aurora Glassmorphism" aesthetic. Backgrounds are seamless, breathing radial gradients in Pastel Pink, Lavender, and Soft Blue. Key Elements: Dozens of 3D Floating React Logos drift through the scene with heavy depth-of-field blur. Rotating dashed "Radar Rings" and "Pulse Circles" add technical texture behind the text. Color Palette: Electric Azure Blue (#0b84f3) for primary accents against Deep Black bold typography (Poppins Black). Motion Physics: Elastic "Layout Smoothing" – new words crash in and physically push existing text aside. Transitions: Elements rotate -15° and shrink to zero to vanish. Scenes switch via massive Blue Iris Wipes and Ring Tunnels. Matches a 140 BPM fast-paced beat.

### 2. Transparent Call-To-Action Overlay
**Link:** https://www.remotion.dev/prompts/transparent-call-to-action-overlay
**Tool:** Claude Code | **Model:** Opus 4.5

**Prompt:** Use remotion best practices. This is our youtube channel: https://www.youtube.com/@remotion_dev. Use curl scrape youtube to find the avatar and the subscriber count. Multiple subscriber counts appear on the page, find the right one. Make a white lower third that slides in from the bottom center. Show the name, subscriber count and avatar. Display a typical fixed width black youtube subscribe button that changes from "Subscribe" to "Subscribed". Use an ease-out animation for pressing in the button and a spring animation with a slight bounce once the button is released. Fade out the lower third. Render it as a transparent prores video.

**Key Techniques:** Transparent ProRes rendering, subscribe button animation, lower third overlay

### 3. Cinematic Tech Intro
**Link:** https://www.remotion.dev/prompts/cinematic-tech-intro
**Tool:** Gemini | **Model:** k2.5

**Prompt:** Create a high-energy Remotion video component (1920x1080, 30fps) for a cinematic "CEO Introduction" featuring Jensen Huang, styled with a Cyberpunk/Tech-Corporate aesthetic. The scene should use a clean white background with a subtle radial noise texture and feature massive, centered text that animates with a dramatic "pop-in" effect (scaling down from 3x to 1x). Introduce a cutout image with an explosive spring entrance, continuous floating motion, and occasional digital glitch distortion (skew/hue-rotate), layered above rotating dashed tech rings and falling Matrix-style data streams. Slide in a Glassmorphism HUD panel from the left (dark semi-transparent with a cut-corner clip-path) that displays bio details using a Monospace font. Enhance depth with a vertical scanner line, animated corner brackets, and floating geometric particles (triangles/hexagons).

**Key Techniques:** Pop-in scaling, glitch distortion, glassmorphism HUD, matrix data streams, corner brackets, geometric particles

> **Note:** This style is good for creating intro pages for presentations

### 4. Three.js "Top 20 Games Sold" Ranking
**Link:** https://www.remotion.dev/prompts/threejs-top-20-games-sold-ranking-1
**Tool:** Claude Code

**Prompt:** Create a Remotion + React Three Fiber visualization of the top 20 best-selling video games as a vertical "tower" using boxes with height representing copies sold. Resolution: 1920x1080, FPS: 60. Animate the camera from the bottom of the last rank to the top of the first rank, stopping on each rank briefly.

**Key Techniques:** React Three Fiber, 3D bar chart, camera animation, data visualization

**Tips from the author:**
1. Include sample dataset up front — paste it in the first prompt
2. Explicitly mention React Three Fiber or three.js for 3D visuals
3. Specify FPS early — changing it later can break animations
4. Include camera movement details in the first prompt
5. Ask Claude to expose values as props for fine-tuning in Studio

### 5. Cursor Agent Skills Announcement
**Link:** https://www.remotion.dev/prompts/cursor-agent-skills-announcement
**Tool:** Cursor

**Prompt:** Create a video project announcing Agent Skills, following brand rules. 1920×1080, 60fps. Typewriter text effect at about 1 character per frame, then hold for 3 seconds after typing on every title card. Line 1 types first, then line 2. Multiple cards with text, then play a screen recording full-screen (framed top-aligned for 2 seconds, then zoom in with easing to 125%). Final card with closing text, then existing end animation full-screen.

**Key Techniques:** Typewriter effect, screen recording integration, continuous zoom with easing, brand rule files

**Notes:**
- Used existing screen recording
- Used existing end card animation
- Had a brand rule file with guidelines and colors
